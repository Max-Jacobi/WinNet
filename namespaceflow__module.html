subroutine, public flow_module::flowcalc ( real(r_kind), dimension(:), intent(in)  Y,
real(r_kind), intent(in)  dt 
)

Flow calculation from jacobian. It is calculated with the help of the Jacobian.

\[ F_{ij} = (J_{ji} \times Y_j - J_{ij} \times Y_i) \Delta t \]

At each timestep the flows are cumulatively added to the flows and/or cum_flows arrays. On each output iteration the flows array and the cumulative timestep is reset to zero.

Note
Using the jacobian directly has the advantage that the flow will be correct if the calculation is correct. In previous versions, the flow was not calculated by using the jacobian.

◆ flowsort()

subroutine, public flow_module::flowsort ( logical, intent(in)  cumulative)

Sort flows and remove zero flows to prepare them for output. Updates the output_* arrays. If cumulative = .true., the cumultive flows are used, otherwise the momentary flows are used. In the second case, the flows array and cum_dt are reset to zero.
